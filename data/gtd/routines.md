# Routines

Recurring commitments that reset on a cycle. Agent resets checkboxes at the start of each period.

## Daily

### Morning
- [ ] Wake up
- [ ] Remind myself: waking up is hard because cortisol builds up and my PFC isn't fully active yet
- [ ] Use my external habit system (alarm) to bypass PFC dependence via basal ganglia/amygdala patterning
- [ ] Tell myself: "It's okay, I can wake up"
- [ ] Brush myself
- [ ] Splash my face
- [ ] Make my bed
- [ ] Open the curtains
- [ ] Stretch myself
- [ ] Use Scribe to plan the day

### Evening
- [ ] Capture anything from today into inbox
- [ ] Prep tomorrow's priorities
- [ ] Close work laptop/machine
- [ ] Note three major insights and celebrate at least one major win in Scribe
- [ ] Say "I am done for the day"
- [ ] Change clothes and/or splash face to signal evening mode
- [ ] Greet family happily!

### Night
- [ ] Brush
- [ ] Splash face

- [ ] Prepare for sleep
- [ ] Do progressive relaxation response meditation
- [ ] Do body scan meditation
- [ ] Smile and thank God and sleep
